Guía de estilos

Tipografía

Mediante las variables se pueden definir los aspectos de la tipografía:

$baseFontFamily = 'Droid Serif', serif
$baseHeadingFontFamily = 'Droid Serif', serif
$iconFontFamily = 'icon'
$fontColor = #111
$fontSize = 16px

En el archivo base/fonts.styl se cargan las fuentes externas mediante @font-face
